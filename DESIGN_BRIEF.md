# Ex-Machina — Дизайн-бриф для фронтенда

## Что это
SaaS-платформа для создания AI-ботов для отелей. Продажник заполняет опросник прямо на встрече с клиентом → демо-бот готов за 5 минут → клиент видит работающий продукт.

Домен: **exmachina.kg**

## Бренд
- Название: **Ex-Machina** (Ex — заглавная)
- Вайб: премиальный tech, уверенный, минималистичный
- Лого: текстовое — `Ex` белый + `-Machina` синий (#3B82F6)
- В sidebar: `EM` в чёрном квадрате

## Структура лендинга (5 секций)

### 1. Hero (тёмный фон)
- Layout: 2 колонки на десктопе
- Слева: badge с social proof → заголовок → описание → 2 кнопки (CTA + войти)
- Справа: мокап телефона с примером чата бота (реальные сообщения Ton Azure)
- Мобильно: мокап скрывается, только текст

### 2. Как это работает (светлый)
- 3 колонки с иконками-цифрами
- Шаг 1: Заполняете опросник (5 мин)
- Шаг 2: Демо-бот готов мгновенно
- Шаг 3: Бот работает 24/7

### 3. Возможности (neutral-50 фон)
- 6 карточек в 3 колонки (2 на мобильном)
- Белые карточки с border, иконки-эмодзи

### 4. Кейс Ton Azure (светлый)
- Карточка по центру, 3 метрики: 90+ диалогов, 86% автоматизация, 3 языка

### 5. Pricing + Footer (тёмный)
- Одна карточка по центру: $20/мес + $700 разово
- Чеклист с синими галочками
- CTA кнопка на всю ширину
- Footer: © 2026 Ex-Machina · exmachina.kg

## Страницы приложения

### Визард создания бота (/hotels/new)
- 3 шага с прогресс-баром
- Слева форма, справа живой preview чата
- Кнопка "Создать демо-бота" → редирект на /demo

### Демо-страница (/hotels/[id]/demo)
- Тёмный фон (#0A0A0A)
- По центру мокап телефона (белый, скруглённый)
- Внутри — реальный чат с AI (через API /preview-chat)
- Кнопка "В админку" сверху

### Dashboard (/dashboard)
- Sidebar слева: EM лого, меню, выход
- Карточки: заявки, активные боты, всего отелей

### Отели (/dashboard/hotels)
- Таблица с колонками: отель, статус (демо/активен), бюджет-бар, диалоги, AI расход
- Бюджет-бар: зелёный (<70%), жёлтый (70-90%), красный (>90%)
- Ссылки: Управление, Демо

### Управление отелем (/dashboard/hotels/[id])
- Карточка "Бюджет": бар + input для изменения лимита
- Карточка "Статус": 3 кнопки (демо/активен/приостановлен)
- Карточка "Каналы": поля для Telegram token + WhatsApp API key
- Карточка "Информация": модель, стиль, языки, дата создания
