**ВАЖНО - Как работает генерация:**

1. Пользователь выбирает стиль → onChange
2. `generatePrompt(formData)` вызывается
3. Textarea обновляется с новым промптом
4. Пользователь видит результат мгновенно
5. Может редактировать вручную если нужно

**Код:**
```typescript
const [formData, setFormData] = useState({...});
const [customPrompt, setCustomPrompt] = useState('');

// Автогенерация при изменении данных
useEffect(() => {
  const generated = generatePrompt(formData);
  setCustomPrompt(generated);
}, [formData.name, formData.rooms, formData.communicationStyle]);

// Пользователь может редактировать
<textarea
  value={customPrompt}
  onChange={(e) => setCustomPrompt(e.target.value)}
/>
```

## 🎯 Ключевые UX принципы

1. **Мгновенная обратная связь:**
   - Промпт генерируется live
   - Preview обновляется в реальном времени
   - Валидация на onChange

2. **Прогресс всегда виден:**
   - Progress bar в мастере
   - Индикаторы загрузки
   - Анимации переходов

3. **Ошибки понятные:**
   - "Введите название отеля" вместо "Field required"
   - Показывать где ошибка
   - Как исправить

4. **Можно вернуться:**
   - Кнопка "Назад" в мастере
   - Данные не теряются
   - Можно редактировать потом

5. **Ничего не выдумываем:**
   - Все данные из формы
   - Генератор промптов прозрачный
   - Можно редактировать результат
